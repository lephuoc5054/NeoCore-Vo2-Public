## 🧹 Common issues

### The app does not open
Try these steps:

- Right-click the app and choose **Run as administrator**
- Check that the file finished downloading
- Make sure you extracted the ZIP first
- Restart Windows and try again

### The AI does not reply
Check:

- Your internet connection
- Your Ollama app, if you use local AI
- The model name in the settings
- The API address in the app

### The HUD looks too small or too large
Open the app settings and change:

- Window size
- Scale
- Font size
- Panel position

### The app cannot find Ollama
Check that:

- Ollama is running
- The local address is correct
- No other app is using the same port
- The model is loaded
